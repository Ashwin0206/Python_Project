Overview :

PAYPULSE is a console-based digital payment system developed using Python and MySQL.
It simulates a basic online banking and payment platform where users can create accounts, log in securely, and perform financial transactions such as mobile recharge, bill payments, deposits, and balance checking.

This project demonstrates database connectivity, authentication systems, and real-time transaction handling using Python.

Technologies Used:

Python

MySQL

mysql.connector

Features:

Account Creation – Allows new users to register with a unique username, password, and 16-digit card number.

Login Authentication – Verifies user credentials before granting access to banking services.

Mobile Recharge – Enables users to recharge prepaid mobile plans with different data pack options.

Electricity Bill Payment – Allows users to pay electricity bills by entering state and consumer number.

Broadband Bill Payment – Generates a random broadband bill amount and processes payment securely.

Money Deposit – Lets users add money to their account balance.

Balance Checking – Displays the current available balance after secure verification.

Transaction History – Stores and displays all transaction records with auto-generated transaction IDs.

Transaction Deletion – Allows users to delete a selected transaction from history.

Database Integration – Connects Python with MySQL to store and update account and transaction data.
